# Basic NPM scripts starter template

Starter template for building simple static sites. Requires Node and NPM (or yarn).

<!-- - Autoprefixer -->
- SCSS
- ESLint

## Step 1) Install dependencies

```

npm i

```

## Step 2) Start Server and start building

```

npm start

```
